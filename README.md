# FEVER-CAPITAL

## ゲーム概要 (Game Overview)
**物理演算 × 悪魔の負債 × マネーロンダリング**

『FEVER CAPITAL』は、物理エンジンの「予測不能な挙動」をギャンブルに変換し、10億の借金を背負いながら毎ターン加速する悪魔の取り立てから生き延びる、物理特化型ローグライク・フィーバーゲームです。全4名のプログラマーチームの開発力を結集し、アセットの質ではなく「ロジックと手触りの狂気」で中毒性を生み出すことをコンセプトとしています。

### 基本ゲームサイクル (Game Loop)
プレイヤーは以下の3つのステップを1ターンとして繰り返し、膨らみ続ける負債を返済します。

1. **仕入れ (UFO Catcher)**
   アームを操作し、筐体内のコインや宝石を物理的に掴み取ります。稀に出現する救済アイテムもここで獲得します。
2. **増幅 (Double or Nothing)**
   獲得したチップを賭け、5種類の物理ミニゲームに挑戦します。成功すれば獲得額が倍増しますが、失敗すればすべてを失います。
3. **洗浄 (Pinball)**
   獲得した「汚れた金」を自作のピンボール台に投入します。釘にバウンドさせることで「洗浄」し、所持金として確定（増殖）させます。

### 物理ミニゲーム詳細
増幅フェーズで挑戦する、物理エンジンを主体とした5つの競技です。
- **鉄球落とし**: V字レールの広がりを調整し、真下のボーナス穴へ。欲張ると手前で脱落。
- **慣性ストップ**: 高速スライダーに摩擦ブレーキをかけ、停止線で止める。質量と速度が毎回変化する。
- **バランス・スタック**: 不安定な土台に物体を自由落下で積み上げる。物理演算の限界まで高さを競う。
- **振り子インパクト**: 重力に従う振り子をタイミング良く放し、目標を弾く。
- **跳ね返りカップ**: 反発係数の高いボールを一度バウンドさせカップへ入れる。

### マネーロンダリング・ピンボール
獲得したチップを洗浄・増殖させる最終工程です。
- **カスタマイズ**: お金を払って「釘」を自由な位置に追加設置可能。
- **分裂の釘**: 当たったコインが物理的に2つにコピーされる。
- **ワープの釘**: コインを上部へ戻し、連鎖バウンドを誘発する。

### 悪魔の取り立てと成長
毎ターンの終わりに、指数関数的に増加する請求額が提示されます。
- **取り立て**: 1円でも足りなければ「最終審判」へ移行。
- **アップグレード**: 返済後の余剰金で、アームの摩擦力強化、釘の在庫追加、利子率の引き下げ（ハック）などを行う。

### 特殊アイテム（救済の鍵）
- **逆行の砂時計**: タイム・リバース。支払いに失敗し、最終審判（ゲームオーバー）が確定した瞬間に発動可能。現在のターンの最初（UFOキャッチャーの直前）まで、物理状態と所持金を巻き戻します。

### 最終審判 (ゲームオーバー)
支払いに失敗した際のフローです。
- **救済確認**: 「逆行の砂時計」を所持している場合、やり直すか選択します。
- **絶望**: アイテムがない、あるいは使用を拒否した場合、悪魔が直接介入。カメラが激しく揺れ、悪魔の手が画面を握りつぶし、視界が完全に真っ暗（Blackout）になり強制終了します。
