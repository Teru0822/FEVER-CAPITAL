# FEVER-CAPITAL

## ゲーム概要 (Game Overview)
**物理演算 × 悪魔の負債 × マネーロンダリング**

『FEVER CAPITAL』は、物理エンジンの「予測不能な挙動」をギャンブルに変換し、10億の借金を背負いながら毎ターン加速する悪魔の取り立てから生き延びる、物理特化型ローグライク・フィーバーゲームです。
全4名のプログラマーチームの開発力を結集し、アセットの質ではなく「ロジックと手触りの狂気」で中毒性を生み出すことをコンセプトとしています。

### 開発方針（ブラッシュアップの3軸）
本企画はプログラマー4名の適性と規模に合わせ、以下の3軸をベースに設計されています。

1. **開発規模の最小化（ワンシーン・アーキテクチャ）**:
   3D空間を「デスク上」に完全固定し、シーン遷移を排除。カメラの切り替えだけでゲームを進行させます。背景やキャラクターモデルの制作コスト（ビジュアル制作コスト）を極限まで削り、ポストエフェクト（見た目の加工）の徹底により「高級感」を演出します。
2. **拡張性**:
   ミニゲームを「モジュール（部品）」化し、スクリプトを追加するだけで新しいギャンブルを無限に増やせる設計です（コードの再利用性の向上）。さらにピンボールの「釘（Peg）システム」をカスタマイズ可能にし、プレイヤー自ら稼げる配置を研究するパズル要素を持たせています。
   これらの仕様により、4人のプログラマーがそれぞれ「キャッチャー担当」「ミニゲーム担当」「ピンボール担当」「システム・UI担当」として完全に並行開発できる体制を実現しています。
3. **収益性・中毒性（永続アップグレード）**:
   「徳ポイント」による永続強化（メタプログレス）を導入。数値調整（パラメータ）でゲーム体験を深め、一度のプレイで負けても「次はもっと楽に稼げる」というローグライク特有のループと強力な「もう一回」のリプレイモチベーションを生み出します。

## 基本ゲームサイクル (Game Loop)
プレイヤーは以下の3つのステップを1ターンとして繰り返し、膨らみ続ける負債を返済します。

1. **仕入れ (UFO Catcher)**
   アームを操作し、筐体内のコインや宝石を物理的に掴み取ります。稀に出現する救済アイテムもここで獲得します。
2. **増幅 (Double or Nothing)**
   獲得したチップを賭け、モジュール化された物理ミニゲームに挑戦します。成功すれば獲得額が倍増しますが、失敗すればすべてを失います。
3. **洗浄 (Pinball)**
   獲得した「汚れた金」を自作のピンボール台に投入します。釘にバウンドさせることで「洗浄」し、所持金として確定（増殖）させます。プレイヤーは状況に合わせ最適な釘の配置を模索します。

## 物理ミニゲーム詳細（拡張モジュール）
増幅フェーズで挑戦する、物理エンジンを主体とした競技群です。スクリプト追加でゲーム全体への容易な拡張が可能です。
- **鉄球落とし**: V字レールの広がりを調整し、真下のボーナス穴へ。欲張ると手前で脱落。
- **慣性ストップ**: 高速スライダーに摩擦ブレーキをかけ、停止線で止める。質量と速度が毎回変化する。
- **バランス・スタック**: 不安定な土台に物体を自由落下で積み上げる。物理演算の限界まで高さを競う。
- **振り子インパクト**: 重力に従う振り子をタイミング良く放し、目標を弾く。
- **跳ね返りカップ**: 反発係数の高いボールを一度バウンドさせカップへ入れる。

## マネーロンダリング・ピンボール
獲得したチップを洗浄・増殖させる最終工程です。
- **カスタマイズ**: お金を払って「釘」を自由な位置に追加設置し、独自のマネーロンダリング機構を構築可能。
- **分裂の釘**: 当たったコインが物理的に2つにコピーされる。
- **ワープの釘**: コインを上部へ戻し、連鎖バウンドを誘発する。

## 悪魔の取り立てとメタプログレス
毎ターンの終わりに、指数関数的に増加する請求額が提示されます。
- **取り立て**: 1円でも足りなければ「最終審判」へ移行。
- **徳ポイント (メタプログレス進化要素)**: ゲームオーバー時に精算されるポイント。これを用いた永続強化により、次回プレイ時のアーム性能強化、釘の初期所持数増加、利子率低下などをアンロックします。「次はもっと楽に稼げる」というローグライク進行を担います。

## 特殊アイテム（救済の鍵）
- **逆行の砂時計**: タイム・リバース。支払いに失敗し、最終審判（ゲームオーバー）が確定した瞬間に発動可能。現在のターンの最初（UFOキャッチャーの直前）まで、物理状態と所持金を巻き戻します。

## 最終審判 (ゲームオーバー)
支払いに失敗した際のフローです。
- **救済確認**: 「逆行の砂時計」を所持している場合、やり直すか選択します。
- **絶望**: アイテムがない、あるいは使用を拒否した場合、悪魔が直接介入。カメラが激しく揺れ、悪魔の手が画面を握りつぶし、視界が完全に真っ暗（Blackout）になり強制終了します。
